Instructions | Mobilis XHunt

This game is the real-time version of the famous board game Scotland Yard.

here are two kinds of characters: Mister X, a criminal, and some agents that haunt him. Mr. X tries not to be caught. Everyone only has a certain number of different tickets, for example 5 bus tickets, 5 tram tickets and 2 railway tickets.

The game consists of 10 rounds, in each one first Mr. X has to choose his next target and then the agents. After this it needs some time until everybody reaches the target station by bus, tram or railway. If everybody arrived and Mr. X is not on the same station as one agent, the next round starts.

If Mr. X is caught the agents win, but if the ten rounds are over and Mr. X wasn't found, he wins.
